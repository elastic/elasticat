## What's New

- Split into two separate binaries: `elasticat` and `catseye`
  - `elasticat`: CLI tool for command-line operations (logs, metrics, traces, search, etc.)
  - `catseye`: Interactive TUI application for browsing and exploring observability data
  - Each binary can be installed independently based on your needs

## Bug Fixes

- None

## Breaking Changes

- The TUI is no longer part of the `elasticat` binary and is now available as `catseye`
